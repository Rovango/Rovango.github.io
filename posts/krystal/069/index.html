车车之之的6月阅读记录

 2020-06-30 1064 字 3 分钟 krystal diary 

接着上次写的《车车之之的5月阅读记录》，这次来记录下6月的阅读情况。我这种起标题无力党也是懒，标题格式都不用改了。

6月继续用小花生App记录，一天没落，在此特别感谢我们家优秀的贾阿姨，认同我的教育理念并配合执行，形成了每天我念中英文书阿姨念中文书的完美分工。

6月一共读过61本/套书，373本/套次，有的时候每天念的太多了，有的书反复念N遍也懒得记了，数据仅作为参考。

以下是6月的读书随记：

# 因为精细动作的发育，本月对翻翻书和机关书的兴趣更加浓重。冰冰和波波依旧是最爱的之一。open the barn door是意外之喜，这本原是我凑单买的，结果收到出乎意料的好，每页都是农场里的动物，对应动物的叫声，还带翻翻小机关，大小非常合适宝宝的手掌，车车之之的小手一把就能翻起来，对小宝宝非常友好。

# 对书中主人翁是baby的书表现出强烈兴趣，提名where is baby’s belly button，正好这本还是翻翻书，因为车车之之太喜欢了，我又去收了Karen katz的其他几本，比如where is baby’s mommy和peek a baby。中文的提名十个手指头和十个脚趾头。

# 对发声书的爱意不减，乐乐趣的几本我会念童谣我会唱英文儿歌，快被翻烂了，搞的我天天跟在少爷们屁股后面给书打补丁。

# 蒲蒲兰的经典三件套蹦、哇、谁的声音，依旧爱到不行，每本都是“连续念N遍都很捧场”系列。

# brown bear已经看够了。caterpillar还凑合。小兔子bunny也看够了。

# 触摸书摸够了。

# 几本洞洞书，比如米菲和peek a who，因为大小非常合适车车之之抓拿翻，所以热情还在。

# 小鸡球球还爱着。依旧最爱藏猫猫藏猫猫哇和答应的真好，也是因为是机关书的原因，可以互动。

# 几本低幼入门的兴趣也不错，喂哎、连在一起。

# 咿咿呀呀已经被我抛弃了，理由见5月读书记录一文。

# 多多英语第一册15本里面有几本可以念的，比如I love fruit、mom and baby之类的，很低幼，质量也好，不怕撕。

# 小狗帕比手偶书已经落灰很久了。我在想，为了对得起我的银子，要不要把手偶单独拆下来……

#本月老母亲最爱的是童立方的一套你好色彩，两辑八本，颜色实在是太太太棒了，内容也好，每次翻到最后都有惊喜。本来想收英文原版，没找到购买渠道，收了中文版，是黑龙江美术出版社的，也不错。

暂时想到这么多。

顺便说个趣事，前几天车车之之脖子起痱子，大概是晚上睡觉的时候热的，虽然室温一直24度，我都冷到盖厚被子……

有一次阿姨口出金句：“你俩怎么这么怕热呢，要不去南极找冰冰和波波吧。”

哈哈哈哈。

这里还是给我一直难产的养育类书单推荐……
